## CSGO World Ranking (HLTV.org)

### This project will consume HLTV.org data to get csgo world ranking teams.

To run this project, you need: 
- NodeJS >= 10
- NPM >= 6

To change this project:
1. You need to clone the same in a folder on your workstation.
2. Make sure if Node and Npm is installed
3. Inside the project root, run `npm install && npm run start`

In the project directory, you can run:

### `yarn start` or `npm run start`

Runs the app in the development mode.<br />
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

The page will reload if you make edits.<br />
You will also see any lint errors in the console.

### `yarn build` or `npm run build`
